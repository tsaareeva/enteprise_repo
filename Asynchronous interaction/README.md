## Установка RabbitMQ 

### Windows:
1. Скачайте установщик: https://rabbitmq.com/install-windows.html
2. Установите RabbitMQ
3. Запустите службу: `rabbitmq-server`

### macOS:
```bash
brew install rabbitmq
brew services start rabbitmq
